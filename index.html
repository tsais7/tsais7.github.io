Figured I would write one of those blogs, typing my thoughts into the void. This would be a good opportunity to start writing, never been strong suit of mine.

Org Mode

I've been using GNU Emacs for some time now, but never touched Org Mode, just didn't have a compelling reason to do so. But I've always wanted a blog, and what better ways to do it other than using Org?

Org is incredible, and I've barely scratched the surface here. It doesn't get in my way, and Emacs being self documenting C-h m makes it trivial to ease into org. Ever since, I noticed how so many modern notetaking/"productivity" apps that tries so hard to only mimic a fraction of Emacs/Org's power. Pathetic and bloat. Electron was a mistake.
